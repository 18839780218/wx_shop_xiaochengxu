# wx_shop_xiaochengxu
小程序商城上传,购物车,微信支付,地址管理,商品管理,优惠劵,运费等功能
### 目录结构：
- images — 存放项目中使用的图片及图标
- pages — 存放项目页面相关文件
- style — 存放独立wxss样式文件，可import引入
